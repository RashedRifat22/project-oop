# Student Result and Attendance Management System

This is a C# project that demonstrates an **Object-Oriented Programming (OOP) based Student Result and Attendance Management System**.

## Features:
- Uses **Abstract Class** (`Person`) and **Interface** (`IReport`).
- Stores **student marks** for OOP, DLD, and CA subjects.
- Calculates **Average CGPA** based on marks.
- Tracks **attendance percentage**.
- Supports **copy constructor** for Student objects.
- Generates **student reports**.
